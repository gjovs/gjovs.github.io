## 关于我

我做那些让一切正常运转的"无聊"工作——微服务、支付网关、审计日志。我曾参与与 **Uber** 和 **Correios** 等大型企业对接的 B2B 项目，目前在 **ZapSign**（**Truora** 旗下公司）担任软件工程师，专注于优化电子签名服务的代码库。

白天写 Node.js、Python，偶尔也写 Go 服务。晚上不是在看货币政策报告，就是在盯着交易所的订单簿，再不然就是在为第四声发音而苦恼。

持有 **PCI-DSS** 认证，深耕支付领域。我构建过直连收单机构的集成、自定义网关路由，以及那种能让凌晨三点的故障稍微不那么痛苦的可观测性系统。
